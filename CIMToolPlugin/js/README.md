## svg-pan-zoom

The svg-pan-zoom javascript library is used by the real-time profile preview ViewPart class. Due to stricter security requirements and running CIMTool in an air-gapped environent. The release currently hosted in this folder is v3.6.2 which we try to keep aligned with (but does not have to be) that shipped with the release of the PlantUML Eclipse plugin that is also integrated (currently release 1.2.0).
